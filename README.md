# Мальтіпу Ленд 🐶

Односторінковий сайт-довідник про породу мальтіпу у стилі нативного iOS-застосунку.

## Що всередині
- Великий гайд: про породу, годування, догляд, виховання, здоров'я
- Окремі розділи: як вибрати заводчика, боротьба з паразитами
- Галерея фото дорослих мальтіпу та цуценят
- FAQ-акордеон

## Особливості
- Глассморфізм, м'які тіні, великі заокруглення (iOS-aesthetic)
- Світла / темна тема з перемикачем (з повагою до системних налаштувань)
- Адаптивний дизайн, мобільне меню з анімацією
- Анімація появи блоків при прокручуванні (Intersection Observer)
- Без фреймворків і зовнішніх бібліотек

## Структура
- `index.html` — увесь контент в одному файлі
- `styles.css` — спільні стилі та теми
- `script.js` — теми, мобільне меню, акордеон, анімації

## Запуск
Просто відкрийте `index.html` у браузері. Сервер не потрібен.
